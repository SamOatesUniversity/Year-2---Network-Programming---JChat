#############
## JServer ##
#############

To start the server, run ServerStart.bat, this will start a server on the specified port.

Features:

* Multithreaded.
* Tcp and UDP.
* Slap, Kick and Warn functions.
* Easy to use Graphical Interface.

Known Bugs:

*none.
